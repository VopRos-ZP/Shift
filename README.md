# Тестовое задание

## Использованные библиотеки:
- Room &dash; выбран потому что сложная структура данных для хранения пользователей
- OkHttp & Retrofit &dash; выбран потому что это базовая библиотека для запросов в сеть, 
к тому же Ktor предоставляет очень широкий функционал, который здесь не нужен
- Koin &dash; выбран потому что с его помощью можно быстро настроить зависимости 
и он достаточно прост в использовании.
- OrbitMVI &dash; легкая библиотека для паттерна MVI, выбрана для того, чтобы продемонстрировать знание паттерна.
- Coil &dash; показ изображений из сети.
